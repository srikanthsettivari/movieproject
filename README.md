# Pachanli Paanchabharuka Website

Static movie landing page for distributors.

## Files
- `index.html` - main website page
- `style.css` - styling
- `assets/poster.png` - movie poster

## Quick Edit
Open `index.html` and replace:
`your-email@example.com`
with your real contact email.

## Host on Netlify
1. Go to netlify.com
2. Create a free account
3. Click **Add new site** → **Deploy manually**
4. Drag and drop the full `pachanli_website` folder
5. Netlify will generate a live website link

## Host on Vercel
1. Go to vercel.com
2. Create a free account
3. Click **Add New Project**
4. Upload/import these files
5. Deploy
